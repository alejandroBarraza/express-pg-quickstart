# install project dependencies

``` npm install ```



# Config pg databases

### 1) Create .env file in your root folder.
### 2) Complete below with your datases config

### example
```
PORT=5000
PGUSER=postgres
PGHOST=localhost
PGPASSWORD=chiharu100
PGDATABASE=mutual
PGPORT=5432
```
